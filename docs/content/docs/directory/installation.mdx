# Installation

Learn how to install and configure the Workflow Directory in your project.

## Prerequisites

[Content coming soon]

Before installing the Directory, make sure you have:

- A Workflow project set up
- Node.js 18+ installed
- Your preferred package manager (npm, pnpm, yarn, or bun)

## Installation Steps

[Content coming soon]

### Step 1: Install the Directory Package

```bash
# npm

# pnpm
pnpm add @workflow/directory

# yarn
```
